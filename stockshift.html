1. Create a ClickUp automation: "When webhook received → If qty < threshold → Send SMS"
2. Or use Zapier: Webhook → Filter (qty < X, date between Mar 1–Aug 15) → SMS by Twilio
3. Paste the webhook URL above — StockShift sends product, location, quantity, and timestamp on every transaction.
